Guess the Number


Fun time now!

There are many games that are developed using python Today we’ll develop a classic game “Guess the Number” using python.


About the game

•  This is a game in which the application generates a number in the range of 1 to 20
•  Then the user is given three chances to guess the exact number.
•  The user is also provided with hints whether the generated number is higher or less than the entered number.
•  If the user finds the number in less than three chances, the user wins •  If the user runs out of chances, the game will finish.


Implementation

Initially, we need to think of generating a number.
•  Python has an inbuilt module called random, and there is a function called randrange() which helps in generating a number.
•  randrange(start, end): this generates a number with the range of start to end both inclusive.
•  This function is used to generate a number in the range of 1 to 20

startGame():

•  This is the function where the complete game is controlled over.
•  Initially, we will generate the number using number_generation() function
number_generation() : This function returns a random number generated from the range 1 to 20 using randrange() function present in the random module
•  The number is returned to the generated_number present in the startGame()
•  We will initialise the variables 

number_of_chances,current_chance

and guessed_number.
•  The actual interaction with the user starts with this step.
•  We will run the following steps until the user utilised three chances, or the user might have guessed the number correctly.
•  We will take the input from the user
•  If it is correct in the first guess, we will display WOW! We got it right in our first guess itself and ask the user if he wishes to play again.
